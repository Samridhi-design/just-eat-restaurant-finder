# Just Eat Restaurant Finder

A React + TypeScript app that fetches restaurant data from the Just Eat API using a UK postcode and displays the first 10 restaurants with the required fields:

- Name
- Cuisines
- Rating
- Address

## Features

- Search restaurants by UK postcode
- Displays only the first 10 restaurants returned
- Shows only the required restaurant data points
- Loading state while fetching results
- Error handling for failed API requests
- Clean and responsive interface

## Tech Stack

- React
- TypeScript
- Vite
- CSS
